# Listas-de-filmes

Projeto simples de site para organizar listas de filmes.  
Feito com **HTML5, CSS3 e JavaScript puro (vanilla JS)**, sem backend por enquanto.

🔗 **Acesse o site aqui:** [Listas de Filmes](https://jheansrosas.github.io/Listas-de-filmes/)

---

## 🚀 Funcionalidades

- Exibição de uma lista de filmes (nome, imagem, etc.).  
- Interação via JavaScript para manipular a página (ex: adicionar ou filtrar filmes).  
- Layout básico e responsivo simples.  

---

## 🛠 Tecnologias usadas

- **HTML5**  
- **CSS3**  
- **JavaScript (vanilla)**
